1. Create two topics:

docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic Numbers --partitions 10
 
docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic CountedNumbers --partitions 10
 
 
2. Run examples.streams.CountNumbersExample

  The application must be running
  
3. Run examples.streams.CountNumbersExampleDriver


4. Check the topic CountedNumbers

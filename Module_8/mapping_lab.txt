1. Create two topics:

 docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic TextLinesTopic --partitions 10
 
docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic UppercasedTextLinesTopic --partitions 10
 
2. Run examples.streams.MapFunctionLambdaExample

  The application must be running
  
3. Run examples.streams.MapFunctionLambdaDriver or

  produce some values with the console producer
  
  docker exec -ti kafka1 /usr/bin/kafka-console-producer --bootstrap-server kafka1:19092 --topic TextLinesTopic
  
4. Check the topic UppercasedTextLinesTopic


 docker exec -ti kafka1 /usr/bin/kafka-console-consumer --bootstrap-server kafka1:19092 --topic UppercasedTextLinesTopic --from-beginning

1. Create two topics:

 docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic LeftStreamTopic --partitions 10
 
 docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic RightStreamTopic --partitions 10

 docker exec -ti kafka1 /usr/bin/kafka-topics --create --bootstrap-server kafka1:19092 --topic JoinedTopic --partitions 10


2. Run examples.streams.JoinStringsExample
  
  The application must be running
  
  
3. Run JoinedStringExampleDriver. This will generate values into LeftStreamTopic and RightStreamTopic


4. Check the topic JoinedTopic

1. Check connector plugins 

curl http://localhost:8083/connector-plugins

2. Check connectors

curl  http://localhost:8083/connectors

2. Create a file sink connector

curl -X POST -H 'Content-Type: application/json' -d @file-connector.json http://localhost:8083/connectors

3. Check connectors 

curl  http://localhost:8083/connectors

4. Check connector details

curl  http://localhost:8083/connectors/file-sink-connector

5. Check the connector tasks

curl http://localhost:8083/connectors/file-sink-connector/tasks

6. Check the status

curl http://localhost:8083/connectors/file-sink-connector/status

7. Produce the values

./kafka-console-producer --topic connect-sink --bootstrap-server localhost:9092 

{"a1":"value1", "a2": "value2"}
{"a1":"value4", "c2": "value3"}

8. Check the file 

./zk/connect/msg.txt

9. Pause the connector

curl -X PUT http://localhost:8083/connectors/file-sink-connector/pause


10. Check the status

curl  http://localhost:8083/connectors/file-sink-connector/status

11. Add more values

{"z1":"value5", "z2": "value6"}
{"z3":"value7", "z4": "value8"}

12. Check the file 

./zk/connect/msg.txt

13. Resume connector

curl -X PUT http://localhost:8083/connectors/file-sink-connector/resume

14. Check the status 

curl  http://localhost:8083/connectors/file-sink-connector/status

15. Check the file 

./zk/connect/msg.txt
